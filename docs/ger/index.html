kbd

Der kbd-Tag kann genutzt werden, um Tastaturabkürzungen darzustellen: STRG + C

samp

Mit dem samp-Tag sollen Konsolen-Ausgaben dargestellt werden. Mit der Zusatzklasse .console erscheint automatisch ein ">" vor dem Tag:

@root: --username "nap" User not found

Spacer

Mit den Klassen m-*, sowie ml-*, mr-*, mt-*, mb-*, mx-* und my-* können Abstände erzeugt werden. Diese Klassen gibts auch für's Padding mit p-* ...

Ein paar Beispiele:

ml-1 pb-3
mx-3 pt-2
my-5 p-5
